1、集合(Set)：不能添加重复元素

2、典型应用：
    1）客户统计
    2）词汇量统计

3、集合的时间复杂度分析
                LinkedListSet      BSTSet

add             O(n)                O(h)

contains        O(n)                O(h)

remove          O(n)                O(h)


理想情况下：满二叉树：h层一共有2^h-1个节点, h = log2(n+1) = O(logn)
最坏情况下：二分搜索树退化成为链表，最差复制度接近O(n)

平均情况下：O(logn)
最差情况下：O(n)

4、有序集合和无序集合
    1）有序集合中的元素具有顺序性，一般基于搜索树实现
    2）无序集合中的元素没有顺序性，基于哈希表的实现

5、多重集合：集合中的元素可以重复
